# FastGuard 系统安装及使用说明

## 1. 系统要求

- **操作系统**：Windows 7/8/10/11
- **处理器**：至少 Intel Core i3 或同等性能处理器
- **内存**：至少 4GB RAM
- **存储空间**：至少 1GB 可用空间
- **Python**：需要安装 Python 3.7 或更高版本

## 2. 安装步骤

### 步骤 1：解压压缩包
1. 下载并保存 FastGuard 压缩包到您的电脑
2. 右键点击压缩包，选择「解压到当前文件夹」或「解压到 FastGuard」
3. 等待解压完成，得到 FastGuard 文件夹

### 步骤 2：安装 Python（如果未安装）
1. 打开浏览器，访问 [Python 官方网站](https://www.python.org/downloads/)
2. 下载适合您 Windows 系统的 Python 3.7 或更高版本的安装包
3. 运行安装包，勾选「Add Python to PATH」选项
4. 点击「Install Now」完成安装
5. 安装完成后，打开命令提示符（Win+R 输入 cmd 回车）
6. 输入 `python --version` 或 `py --version` 确认 Python 安装成功

### 步骤 3：安装项目依赖
1. 打开命令提示符
2. 使用 `cd` 命令进入 FastGuard 文件夹，例如：
   ```
   cd D:\路径\到\FastGuard
   ```
3. 输入以下命令安装依赖：
   ```
   pip install -r requirements.txt
   ```
4. 等待依赖安装完成

## 3. 运行步骤

### 方法 1：使用批处理文件运行
1. 进入 FastGuard 文件夹
2. 双击 `run_fastguard.bat` 文件
3. 系统会自动启动 FastGuard 应用

### 方法 2：手动运行
1. 打开命令提示符
2. 使用 `cd` 命令进入 FastGuard 文件夹
3. 输入以下命令：
   ```
   python main.py
   ```
4. 按回车运行

## 4. 使用说明

### 基本功能
FastGuard 是一个智能驾驶辅助系统，主要功能包括：
- 目标检测
- 碰撞预警
- 视频处理
- 界面交互

### 操作指南
1. 启动应用后，会显示主界面
2. 根据界面提示进行操作
3. 系统会实时处理视频流并提供预警信息
4. 使用完成后，可直接关闭窗口退出应用

## 5. 常见问题解决

### 问题 1：运行时出现「找不到 Python」错误
- 解决方案：确保已正确安装 Python 并添加到系统 PATH
- 尝试使用 `py -3 main.py` 命令运行

### 问题 2：缺少依赖库错误
- 解决方案：重新运行 `pip install -r requirements.txt` 确保所有依赖安装成功

### 问题 3：应用启动后无响应
- 解决方案：检查系统资源是否充足，关闭其他占用资源的应用
